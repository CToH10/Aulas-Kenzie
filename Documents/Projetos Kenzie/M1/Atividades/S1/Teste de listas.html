Como criar listas ordenadas

  1. Abra o VSCode
  2. Crie um novo arquivo HTML
  3. Utilize a tag "ol"
  4. Liste os itens desejados na ordem correta

Como criar listas não ordenadas

  • Abra o VSCode
  • Crie um novo arquivo HTML
  • Utilize a tag "ul"
  • Liste os itens desejados
